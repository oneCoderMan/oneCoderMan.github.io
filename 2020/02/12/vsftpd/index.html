vsftpd安装和使用
版本：ubuntu16.04与vsftp-2.2.2

0x00 介绍

vsftp(very secure FTP daemon)，一个免费的，开源的ftp服务器软件。
小巧轻快，安全易用，支持虚拟用户，宽带限制。

0x01 安装

apt-get install vsftpd
默认的配置文件在/etc/vsftpd.conf
启动服务service vsftpd start
查看启动日志service vsftpd status
